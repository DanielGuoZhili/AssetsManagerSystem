# 💰 当前收支情报页面

这是一个个人财务管理页面，用于追踪和管理您的订阅服务、投资策略和日常开支。

## 📋 功能特性

### 1. 收支总览
- **当前月收入**：显示您的月收入总额，可点击编辑按钮修改
- **预计月支出**：自动计算所有支出项目的月均值
- **可支配余额**：自动计算收入减去支出的余额

### 2. 已订阅服务
管理您的所有订阅服务，包括：
- 百度网盘
- iCloud
- Cursor
- Amazon
- DC 卡等

每个订阅卡片显示：
- 服务名称
- 金额和货币
- 扣费周期（每月/每年）

### 3. 投资策略
分类管理投资项目：

**A股投资**
- 红利ETF（根据剩余工资灵活投入）

**NISA投资**
- eMAXIS Slim 全世界株式（每月 2万日元）
- 个股投资如任天堂（每月 1万日元）

### 4. 每月花费明细
详细列出所有固定和变动支出：
- 房租：85,000日元
- 年金：约17,100日元
- 保险：约44,000日元
- 住民税：71,000日元/期（每3个月）
- 信用卡还款：80,000-130,000日元
- 电脑分期：4,900日元（至2025年2月）
- 各类订阅服务

## 🎨 界面设计

### 卡片式布局
- 每个项目以独立卡片展示
- 鼠标悬停时卡片会有轻微上浮效果
- 不同类型使用不同颜色图标区分

### 响应式设计
- 在大屏幕上以3列网格展示
- 在中等屏幕上以2列展示
- 在手机上以单列展示

### 颜色编码
- 💚 绿色：收入相关
- 💙 蓝色：订阅服务
- 💛 橙色：日常支出
- 🎯 紫色：投资项目

## ✏️ 编辑功能

### 如何编辑
1. **编辑收入**：点击收入卡片上的编辑图标
2. **编辑订阅/投资/支出**：直接点击对应的卡片
3. **添加新项目**：点击每个部分右上角的"添加"按钮

### 编辑弹窗
每个编辑弹窗包含：
- 名称输入框
- 金额输入框
- 周期输入框
- 备注输入框（可选）
- 保存/删除按钮

### 数据持久化
- 所有数据自动保存在浏览器的 localStorage 中
- 刷新页面不会丢失数据
- 清除浏览器缓存会清空数据

## 💡 使用技巧

### 1. 灵活的金额格式
金额字段支持多种格式：
- 固定金额：`450`
- 金额区间：`80000-130000`
- 文字说明：`根据剩余工资`

### 2. 周期性支出计算
系统会自动处理不同周期的支出：
- 每月支出：直接计入
- 每3个月支出：自动除以3计算月均值
- 每年支出：可手动标注

### 3. 快速分类
使用不同的图标和颜色快速识别：
- ⚡ 闪电图标：订阅服务
- 📈 上升图标：投资项目
- 💰 金钱图标：日常支出

## 🔧 技术实现

### 数据结构
```typescript
interface Subscription {
  id: string
  name: string
  amount: string
  cycle: string
  currency: string
}

interface Investment {
  id: string
  category: string  // A股、NISA
  name: string
  amount: string
  cycle: string
  note?: string
}

interface Expense {
  id: string
  name: string
  amount: string
  cycle: string
  note?: string
}
```

### 本地存储
数据存储在 `localStorage` 的 `financeData` 键中，JSON 格式。

### 自动计算
支出总额会自动计算，考虑不同周期的支出项目。

## 🐛 问题排查

### 数据丢失
如果数据丢失，可能是：
- 浏览器缓存被清除
- 使用了隐私模式
- 更换了浏览器

**建议**：定期截图或手动备份重要数据

### 计算不准确
如果总额计算不准确：
- 检查金额格式是否正确
- 确认周期字段填写正确
- 刷新页面重新计算
